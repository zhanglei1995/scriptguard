## 🏷️ 自动 Status 流转 (Workflows)

设置完以下 Workflow 后，状态会自动变化：

```
   [新 issue 打开]
         ↓ 自动
   ┌─────────┐
   │  Todo   │ ←── Issue 标签 "prio:P0" 自动设置 Priority
   └─────────┘
         ↓ 有人 Assign 自己
   ┌──────────┐
   │In Progress│
   └──────────┘
         ↓ 提交 PR
   ┌──────────┐
   │ In Review│
   └──────────┘
         ↓ PR 合并
   ┌──────────┐
   │   Done   │ ✅
   └──────────┘
```
