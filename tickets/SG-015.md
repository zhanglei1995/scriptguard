**Description**：
Content Script 获取匹配的脚本后，按 `runAt` 时机注入到目标页面。

**Acceptance Criteria**：

- [ ] 支持 `document_start` / `document_idle` / `document_end` / `manual` 四种时机
- [ ] 注入方式：`page.evaluate` 或 `chrome.scripting.executeScript`
- [ ] 注入超时保护
- [ ] 注入失败时记录错误日志
- [ ] 注入成功时记录开始时间
- [ ] 执行完成后自动运行关联规则
