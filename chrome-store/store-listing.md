# ScriptGuard - Chrome Web Store 上架信息

## Store Name

## Short Description (132 chars max)
监控用户脚本健康状态，自动检测变更和异常行为，保护浏览安全。

## Detailed Description
ScriptGuard 是一款用户脚本健康监控与管理平台。通过实时监控脚本的行为变化，帮助用户及时发现潜在的安全风险。

主要功能：
• 脚本健康监控 - 实时检测脚本运行状态和行为变化
• 变更检测 - 自动对比脚本版本，发现未授权修改
• 行为分析 - 分析脚本的网络请求、DOM 操作等行为
• 智能告警 - 发现异常时通过浏览器通知提醒用户
• 脚本管理 - 统一管理已安装的用户脚本
• 云端同步 - 支持多设备间同步脚本配置

适用场景：
• Greasy Fork / Tampermonkey 脚本用户
• 关注脚本安全性的开发者
• 需要管理大量用户脚本的高级用户
